This webiste features 5 elements that demonstrate a variety of features present in babylon.js. This about page will provide any details that may be relevant to using the elements.


Element 4

Controls:

  • W / Up arrow key - Move Forward
  • S / Down arrow key - Move Backwards
  • A / Left arrow key - Move Left
  • D / Right arrow key - Move Right

In element 4, if the player pushes the box off the edge of the ground, it will respawn in the middle of the platform.


Element 5

Controls:

  • W / Up arrow key - Move Forward
  • S / Down arrow key - Move Backwards
  • A / Left arrow key - Move Left
  • D / Right arrow key - Move Right

The goal of element 5 is to collect crates. When the player collides with a crate, it is destroyed and another one will spawn.
